## List of Features

- [x] Dynamic Point Lighting
- [x] Camera System
- [x] Stencil Masking
- [x] Blur Outlining
- [x] Shadow Mapping
- [x] Cubemapping
- [x] Alpha Blending
- [x] Model Selection/Manipulation
